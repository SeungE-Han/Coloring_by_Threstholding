# Coloring_by_Threstholding
Color image by the range of  Thresthold

1. 히스토그램을 그린다.
2. 육안으로 4개의 peak를 확인한다.
3. 임계값을 구한다.
4. 임계값의 범위에 따라 파랑, 초록, 빨강, 노랑 색을 칠한다.
